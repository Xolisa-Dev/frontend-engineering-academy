Frontend Engineering Academy

📖 About

This repository documents my journey toward becoming a stronger frontend engineer through structured learning, practical exercises, and real-world projects.

Rather than focusing only on frameworks, this academy emphasizes the fundamentals that power every modern frontend application.

**Topics include:**

🌐 Browser Fundamentals

📄 DOM Manipulation

⚡ JavaScript

🏗 TypeScript

🎨 CSS Architecture

♿ Accessibility

🚀 Performance

🧪 Testing

🧩 Design Systems

🏛 Frontend Architecture

🎯 Goal

To develop the knowledge, engineering mindset, and technical confidence required to grow from Frontend Developer to Frontend Architect.

📚 Learning Philosophy

_**For every topic I aim to answer:**_

What is it?

Why does it exist?

How does it work?

When should I use it?

How would an architect think about it?
